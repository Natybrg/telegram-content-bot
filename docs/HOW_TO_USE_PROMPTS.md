```

  • פרומפט #1: PeerIdInvalid בערוצים (הבעיה שלך!)
  
🟠 גבוהה:
  • פרומפט #2: Race Condition

  • פרומפט #4: Memory Leak
  • פרומפט #5: ProcessPool

```

### שלב 2: העתק את הפרומפט

1. **פתח** את `IMPLEMENTATION_PROMPTS.md`
2. **מצא** את הפרומפט שבחרת (למשל "פרומפט #1")
3. **העתק** את **כל** הטקסט מתחילת הפרומפט ועד הסוף
   - כולל הכותרת
   - כולל כל החלקים (שלב 1, שלב 2, וכו')
   - עד ה-"```" האחרון

### שלב 3: הדבק ב-AI

1. **פתח** ChatGPT / Claude / Gemini
2. **התחל שיחה חדשה**
3. **הדבק** את הפרומפט המלא
4. **שלח**

### שלב 4: עקוב אחרי ההוראות

ה-AI יתן לך:
- ✅ ניתוח מעמיק
- ✅ קוד לתיקון
- ✅ הוראות בדיקה
- ✅ עדכוני תיעוד

**עקוב אחרי כל ההוראות בסדר!**

### שלב 5: בצע ובדוק

1. **קרא** את התיקון שה-AI נתן
2. **הבן** מה השינוי עושה
3. **בצע** את השינויים בקוד שלך
4. **גבה** את הקוד לפני (Git commit)
5. **הרץ** את הבוט
6. **בדוק** שהכל עובד
7. **תעד** את השינוי

### לפני שמתחילים:

```bash

```

✅ **קרא את הפרומפט:**
לפני שמדביקים, קרא מה הוא מבקש

אל תעתיק קוד שאתה לא מבין

✅ **בדוק אחרי כל שינוי:**
```bash

```

אל תעשה 5 פרומפטים ביחד!
